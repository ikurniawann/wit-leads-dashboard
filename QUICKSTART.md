# WIT Dashboard - Quick Start Guide

**Get started in 5 minutes!** ⚡

---

## 🔗 Access Dashboard

**URL:** https://wit-leads-dashboard.vercel.app

**No login required!** Just open and use.

---

## 📋 First Steps

### **1. View Dashboard** 📊
- **URL:** `/`
- **See:** Real-time stats, charts, recent leads
- **Actions:** Add/Edit/Delete leads

### **2. Add Your First Lead** ➕
1. Click **"Add Lead"** button (top right)
2. Fill in:
   - **Company Name:** e.g., "PT Example"
   - **Project Name:** e.g., "IoT Platform"
   - **Client Name:** e.g., "Example Corp"
   - **PIC:** e.g., "Ilham"
   - **Status:** NEW
   - **Value:** Rp 100,000,000
   - **Probability:** 50%
3. Click **"Create Lead"**

### **3. View Clients** 👥
- **URL:** `/clients`
- **See:** All clients database
- **Actions:** Add/Edit/Delete clients

### **4. Add Employees** 👔
- **URL:** `/employees`
- **Click:** "Add Employee"
- **Fill:** Name, email, position, department
- **Status:** Active/Inactive/Probation

### **5. Check Analytics** 📈
- **URL:** `/analytics`
- **See:** Revenue by PIC, Monthly trends
- **Metrics:** Win rate, conversion, pipeline health

### **6. Export Reports** 📤
- **URL:** `/reports`
- **Choose:** Excel or PDF
- **Select:** Leads, Clients, or Employees
- **Download:** File saves automatically

## 🐛 Troubleshooting

**Page shows 0 data?**
- Data might be empty (seed employees first)
- Check Supabase connection

**Export not working?**
- Install dependencies: `xlsx`, `jspdf`
- Check browser pop-up blocker

**Charts not loading?**
- Check browser console for errors
- Refresh page (Cmd/Ctrl + R)
